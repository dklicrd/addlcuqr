Configuración de Google Sheets

  1. Crea una nueva hoja de cálculo en Google Sheets
  2. Ve a Extensiones > Apps Script
  3. Reemplaza el código por defecto con el script proporcionado
  4. Guarda el proyecto y despliégalo como una aplicación web
  5. Copia la URL de despliegue y pégala en el código JavaScript
